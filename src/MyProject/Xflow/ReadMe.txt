This is a sample Scope project, with Canaveral onboarded.
You can edit your Scope scripts, views, modules in your favoriate editor.
All changes will be automatically deployed to XFlow and Cosmos folder by build CI pipeline.

1) Xflow deployment's configuration is set in /xflowDefinition/*.wfdef files
	- TO deploy the workflow to another Xflow folder, you should create the folder firstly in Xflow and give "redmond\cdsbc" permission to access the folder.

2) Views/Modules deployment's configuration is set in project's "DeployToAdls" tag. (Open project file in text to change it)

For details, please check wiki https://osgwiki.com/wiki/CDML/Repo_Creation
